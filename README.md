# Cardiovascular Disease Prediction using Machine Learning

## 📌 Project Overview

Cardiovascular disease (CVD) is one of the leading causes of death worldwide. Early prediction of heart disease can help in timely diagnosis and treatment. This project uses various Machine Learning classification algorithms to predict the presence of cardiovascular disease based on patient health data.

---

## 🎯 Objective

The objective of this project is to:

- Perform data preprocessing.
- Analyze and visualize the dataset.
- Generate a correlation matrix of all features.
- Train and compare multiple Machine Learning models.
- Identify the best-performing model for cardiovascular disease prediction.

---

## 📂 Dataset

**Dataset Name:** Cardio Train Dataset

The dataset contains patient health information such as:

- Age
- Gender
- Height
- Weight
- Blood Pressure (Systolic & Diastolic)
- Cholesterol Level
- Glucose Level
- Smoking Habit
- Alcohol Consumption
- Physical Activity
- Cardiovascular Disease (Target Variable)

## 📊 Data Preprocessing

The following preprocessing steps were performed:

- Loaded the dataset
- Removed unnecessary ID column
- Converted age from days to years
- Created a new BMI feature
- Checked for missing values
- Checked for duplicate records
- Standardized features before model training

## 🤖 Machine Learning Models

The following classification algorithms were implemented:

- Logistic Regression
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest

---

## 📊 Model Evaluation

The models were evaluated using **Accuracy Score**, and their performance was compared to determine the best-performing model.
